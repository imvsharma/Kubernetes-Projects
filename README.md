# 📖 About

This repository contains **150 real-world Kubernetes projects** designed to help developers learn Kubernetes through **practical DevOps scenarios**.

Instead of theoretical tutorials, this repository focuses on **hands-on cloud-native infrastructure projects** used in real production systems.

The projects are structured in progressive levels:


```
Beginner ==> Intermediate ==> Advanced ==> Expert ==> Pro
```


---

# 🎯 Why This Repository

Many Kubernetes tutorials teach concepts, but very few teach **how to build real systems**.

This repository helps you:

- Learn Kubernetes through **hands-on projects**
- Understand **production infrastructure**
- Build a **DevOps portfolio**
- Prepare for **DevOps / Platform Engineer interviews**
- Master **cloud-native architecture**
